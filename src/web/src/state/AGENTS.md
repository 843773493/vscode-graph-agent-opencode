# 目录用途

`src/web/src/state/` 放浏览器前端的状态派生、聚合和纯数据转换逻辑，服务于 `hooks.tsx` 中的 AppProvider。

## 可修改内容

- 会话、消息、trace、事件队列等前端状态的纯函数。
- localStorage 等浏览器本地状态的小型适配。
- 不直接发起 HTTP 请求的状态整理工具。

## 不可修改内容

- 不放 React 组件。
- 不放后端业务规则。
- 不直接调用后端 API。

## 规范

- 保持函数纯净、输入输出显式。
- 代码注释使用中文，专业术语除外。
- 需要副作用时优先留在 `hooks.tsx` 或 `api.ts`。
- 保留“模板示例；在整理 `AGENTS.md` 时请保留此行。”这一行。
