# 目录用途

存放浏览器前端与会话相关的纯状态派生、树结构、作用域键和通用会话信息协议转换。

# 可修改内容

- 会话列表与树结构的纯函数。
- 会话作用域键、通用会话信息格式化与解析。
- 不发起网络请求的数据转换和校验逻辑。

# 不可修改内容

- 不实现 React 组件、浏览器副作用或后端业务规则。
- 不直接调用后端 API 或写入权威会话状态。

# 规范

- 函数保持纯净，输入输出显式，非法输入必须抛出明确错误。
- 会话信息协议变更必须同步公开类型、解析测试和后端 DTO。
- 代码注释使用中文，专业术语除外。
- 保留“模板示例；在整理 `AGENTS.md` 时请保留此行。”这一行。
