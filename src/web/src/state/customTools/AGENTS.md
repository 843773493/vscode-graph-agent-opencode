# 目录用途

`src/web/src/state/customTools/` 存放前端对固定扩展工具入口协议的纯解析 helper。

# 可修改内容

- 可以维护 `invoke_custom_tool` 相关常量、参数解析、调用链展示名和安全 JSON 解析。
- 可以新增只依赖普通对象输入输出的协议工具函数。

# 不可修改内容

- 不在这里读取后端 API、访问 DOM 或渲染 React 组件。
- 不在这里放具体业务视图的布局逻辑。

# 规范

- 对模型/日志返回的字符串参数必须安全解析，解析失败返回空结构或显式未知值，不抛到 UI 层。
- 保持 helper 无副作用，方便请求日志、Agent State 和事件视图共享。
